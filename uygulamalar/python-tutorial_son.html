10. Ya bundan sonra?

Bu kılavuzu okumak muhtemelen Python kullanmaya olan ilginizi artırmıştır. Peki şimdi ne yapmalısınız?

Standart Python dağıtımı çok zengin bir modül kitaplığı ile gelmektedir. Python yazılımları yazarken zamandan büyük tasarruf sağlayacak modüllerin nasıl kullanıldığını öğrenmek için bunlar ile ilgili belgelere başvurun.

Resmi Python sanalyöresi http://www.python.org/ olup; yazılımlar, belgeler ve internetteki Python ile ilgili diğer sanalyörelere bağlantılar içerir.

Python ile ilgili sorular ve hata raporları için <python-list@python.org> e-posta listesini kullanabilirsiniz. Eposta atmadan önce https://docs.python.org/3/faq/index.html#faq-index adresindeki SSS listesine bakmayı ihmal etmeyin. E-posta listesi arşivleri ise https://mail.python.org/pipermail/ adresinde bulunabilir.

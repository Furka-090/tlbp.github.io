9.1. Karşılaştırma İşlevinin Tanımlanması

Sıralı dizi kütüphanesi işlevlerinin kullanılması sırasında dizi elemanlarının nasıl karşılaştırılacağının belirtilmesi gerekir.

Bunu yapmak için, dizideki iki elemanı karşılaştıracak bir karşılaştırma işlevi yazılımcı tarafından sağlanmalıdır. Kütüphane bu işlevi çağırırken karşılaştırılacak dizi elemanlarını işleve bağımsız değişken olarak belirtecektir. Bu karşılaştıma işlevi strcmp (Dizi/Dizge Karşılaştırması) işlevinin yaptığı gibi bir değer döndürmelidir: İlk bağımsız değişken ikincisinden küçükse negatif, büyükse pozitif değer döner.

Aşağıdaki örnekte, double türünden sayılardan oluşan bir dizi ile çalışan bir karşılaştırma işlevi vardır:

Örnek: 
int
compare_doubles (const void *a, const void *b)
{
  const double *da = (const double *) a;
  const double *db = (const double *) b;

  return (*da > *db) - (*da < *db);
}

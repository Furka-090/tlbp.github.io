6.4.2. Evresel Olmayan Dizge Dönüşümleri

ISO C90 standardında tek karakterlerden başka dizgeleri dönüştürecek işlevler de tanımlanmıştır. Bu işlevler ISO C90 standardınının 1. düzeltmesinde tanımlanmış olan benzerleriyle aynı sorunlardan muzdariptir; bkz. Dizge Dönüşümleri.

      işlev
#include <stdlib.h>
size_t mbstowcs(wchar_t *wstring, const char *string, size_t size)
Evresel Güvenilirlik: | MT-Safe | AS-Unsafe corrupt heap lock dlopen | AC-Unsafe corrupt lock mem fd

mbstowcs işlevi ("multibyte string to wide character string" kısaltmasıdır), string boş bayt sonlandırmalı çok baytlı karakter dizgesini geniş karakter kodlu bir diziye dönüştürür ve sonucun ilk size geniş karakterini wstring içinde saklar. Sonlandıran boş karakter boyuta dahildir. Bu durumda string'den elde edilen geniş karakterlerin sayısı size bayttan fazla ise sonuç sonlandırıcı boş geniş karakteri içermeyecektir.

string'deki karakterlerin dönüşümü ilk öteleme durumuyla başlar.

Çok baytlı karakter dizgesi geçersizse, mbstowcs işlevi -1 değeri ile döner. Aksi takdirde wstring dizisi içinde saklanan geniş karakterlerin sayısı ile döner. Bu değer size bayttan küçükse sonlandırıcı boş karakter dahil değildir.

Aşağıdaki örnekte çok baytlı karakter dizisi dönüşümü ve elde edilen sonuç için nasıl yer ayrılacağı gösterilmiştir.

Örnek: 
wchar_t *
mbstowcs_alloc (const char *string)
{
  size_t size = strlen (string) + 1;
  wchar_t *buf = xmalloc (size * sizeof (wchar_t));

  size = mbstowcs (buf, string, size);
  if (size == (size_t) -1)
    return NULL;
  buf = xreallocarray (buf, size + 1, sizeof *buf);
  return buf;
}

Eğer wstring boş gösterici ise, çıktı yazılmaz ve dönüşüm yukarıdaki gibi devam eder ve sonuç döndürülür. Uygulamada bu tür davranışlar, dizgeyi dönüştürmek için gereken tam geniş karakter sayısını hesaplamak için kullanışlıdır. wstring için boş gösterici kabul etme davranışı, ISO C'de belirtilmeyen ve POSIX'te isteğe bağlı olan bir XPG4.2 uzantısıdır.

      işlev
#include <stdlib.h>
size_t wcstombs(char *string, const wchar_t *wstring, size_t size)
Evresel Güvenilirlik: | MT-Safe | AS-Unsafe corrupt heap lock dlopen | AC-Unsafe corrupt lock mem fd

wcstombs işlevi ("wide character string to multibyte string" kısaltmasıdır), boş geniş karakter sonlandırmalı wstring geniş karakter dizgesini çok baytlı karakter dizgesine dönüştürür ve sonucun size baytını string içinde saklar. Yeterince yer yoksa sonuç sonlandırıcı boş karakteri içermez. Karakter dönüşümü ilk öteleme durumuyla başlar.

Sonlandırıcı boş karakter boyuta dahildir. Bu bakımdan size, wstring içindeki baytların sayısına eşit ya da daha küçükse sonlandırıcı boş bayt saklanmaz.

Kod için geçerli çok baytlı karakter bulunamazsa işlev -1 ile döner. Aksi takdirde, string dizisi içinde saklanan baytların sayısı ile döner. Bu değer size bayttan küçükse sonlandırıcı boş karakter dahil değildir.

26.8. BSD Süreç Bekletme İşlevleri

GNU C Kütüphanesi, BSD ile uyumluluk için wait3 işlevini de sağlar. Bu işlev sys/wait.h başlık dosyasında bildirilmiştir. Daha esnek olan wait4 işlevininin öncülüdür. wait3 işlevi artık kullanılmamaktadır.

      işlev
#include <sys/wait.h>
pid_t wait3(int *status-ptr, int options, struct rusage *usage)
Evresel Güvenilirlik: | MT-Safe | AS-Safe | AC-Safe

Eğer usage boş gösterici ise, wait3 işlevi waitpid (-1, status-ptr, options) işlevinin eşdeğeridir.

Eğer usage boş gösterici değilse, wait3 alt sürecin kullanım şekillerini *usage içinde saklar (eğer alt süreç sonlandıysa; durduysa değil). Bkz. Kaynak Kullanımı.

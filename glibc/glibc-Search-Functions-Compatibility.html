5.9.1. Uyumlu Dizge Arama İşlevleri

      işlev
#include <string.h>
char *index(const char *string, int c)
Evresel Güvenilirlik: | MT-Safe | AS-Safe | AC-Safe

index işlevi strchr işlevinin diğer adıdır; ikisi tamamen aynıdır. index işlevi BSD'dan gelir ve System V'den türetilmiş sistemlerde hiç kullanılmamıştır. ISO C bu ismi değil, strchr ismini içerdiğinden daima strchr kullanılmalıdır.

      işlev
#include <string.h>
char *rindex(const char *string, int c)
Evresel Güvenilirlik: | MT-Safe | AS-Safe | AC-Safe

rindex işlevi strrchr işlevinin diğer adıdır; ikisi tamamen aynıdır. rindex işlevi BSD'dan gelir ve System V'den türetilmiş sistemlerde hiç kullanılmamıştır. ISO C bu ismi değil, strrchr ismini içerdiğinden daima strrchr kullanılmalıdır.

3.2.6.11. Yığınak İşlevlerinin Listesi

Burada yığınaklarla ilgili işlevlerin bir özetine yer verilmiştir. Her biri ilk bağımsız değişken olarak struct obstack * türünde bir yığınak adresi alır.

      işlev
#include <obstack.h>
int obstack_init(struct obstack *obstack-ptr)

Yığınağı ilklendirir. Bkz. Yığınak Oluşturma.

      işlev
#include <obstack.h>
void *obstack_alloc(struct obstack *obstack-ptr, int size)

İlklendirmeden size baytlık nesne tahsis eder. Bkz. Bir Yığınağa Nesne Eklenmesi.

      işlev
#include <obstack.h>
void *obstack_copy(struct obstack *obstack-ptr, void *address, int size)

address'ten kopyalanan içerikle size baytlık nesne tahsis eder. Bkz. Bir Yığınağa Nesne Eklenmesi.

      işlev
#include <obstack.h>
void *obstack_copy0(struct obstack *obstack-ptr, void *address, int size)

address'ten kopyalanan size baytlık içerikle, boş karakter sonlandırmalı size+1 baytlık nesne tahsis eder. Bkz. Bir Yığınağa Nesne Eklenmesi.

      işlev
#include <obstack.h>
void obstack_free(struct obstack *obstack-ptr, void *object)

object nesnesini (ve object nesnesinden sonra tahsis edilen tüm nesneleri) serbest bırakır. Bkz. Yığınaktan Nesne Çıkarılması.

      işlev
#include <obstack.h>
void obstack_blank(struct obstack *obstack-ptr, int size)

Büyüyen nesneye ilklendirilmemiş size bayt ekler. Bkz. Büyüyen Nesneler.

      işlev
#include <obstack.h>
void obstack_grow(struct obstack *obstack-ptr, void *data, int size)

Büyüyen nesneye data adresinden kopyalanan size baytı ekler. See Büyüyen Nesneler.

      işlev
#include <obstack.h>
void obstack_grow0(struct obstack *obstack-ptr, void *data, int size)

data adresinden kopyalanan size baytlık içeriği ve sondaki boş karakter için 1 bayt daha ekler. Bkz. Büyüyen Nesneler.

      işlev
#include <obstack.h>
void obstack_1grow(struct obstack *obstack-ptr, char c)

Her seferinde 1 bayt eklemek için kulllanılır. Büyüyen nesneye c karakterini içeren tek bir bayt ekler. Bkz. Büyüyen Nesneler.

      işlev
#include <obstack.h>
void obstack_ptr_grow(struct obstack *obstack-ptr, void *data)

Nesneye gösterici ekler. data değerini içeren sizeof (void *) baytı ekler. Bkz. Büyüyen Nesneler.

      işlev
#include <obstack.h>
void obstack_int_grow(struct obstack *obstack-ptr, int data)

Nesneye int türünde tek bir değer ekler. Büyüyen nesneye sizeof (int) ekler ve data değeri ile ilklendirir. Bkz. Büyüyen Nesneler.

      işlev
#include <obstack.h>
void *obstack_finish(struct obstack *obstack-ptr)

Büyüyen nesneyi bitirir ve kalıcı adresini döndürür. Bkz. Büyüyen Nesneler.

      işlev
#include <obstack.h>
int obstack_object_size(struct obstack *obstack-ptr)

Büyüyen nesnenin geçerli boyutunu döndürür. Bkz. Büyüyen Nesneler.

      işlev
#include <obstack.h>
int obstack_room(struct obstack *obstack-ptr)

Geçerli nesnenin büyütülmesinde kullanılabilecek oda miktarını döndürür. Bkz. Çok Hızlı Büyüyen Nesneler.

      işlev
#include <obstack.h>
void obstack_1grow_fast(struct obstack *obstack-ptr, char c)

Yeterli oda olduğuna bakmaksızın büyüyen nesneye c karakterini içeren tek baytı ekler. Bkz. Çok Hızlı Büyüyen Nesneler.

      işlev
#include <obstack.h>
void obstack_ptr_grow_fast(struct obstack *obstack-ptr, void *data)

obstack-ptr yığınağındaki büyüyen nesneye data değerini içeren sizeof (void *) baytı ekler. Bkz. Çok Hızlı Büyüyen Nesneler

      işlev
#include <obstack.h>
void obstack_int_grow_fast(struct obstack *obstack-ptr, int data)

obstack-ptr yığınağındaki büyüyen nesneye data değerini içeren sizeof (int) baytı ekler. Bkz. Çok Hızlı Büyüyen Nesneler

      işlev
#include <obstack.h>
void obstack_blank_fast(struct obstack *obstack-ptr, int size)

Yeterli oda olduğuna bakmaksızın büyüyen nesneye ilklendirilmemiş size baytı ekler. Bkz. Çok Hızlı Büyüyen Nesneler.

      işlev
#include <obstack.h>
void *obstack_base(struct obstack *obstack-ptr)

Büyümekte olan nesnenin geçici başlangıç adresini döndürür. Bkz. Bir Yığınağın Durumu.

      işlev
#include <obstack.h>
void *obstack_next_free(struct obstack *obstack-ptr)

Büyümekte olan nesnenin sonundaki sonraki adresi döndürür. Bkz. Bir Yığınağın Durumu.

      işlev
#include <obstack.h>
int obstack_object_size(struct obstack *obstack-ptr)

Büyümekte olan nesnenin bayt cinsinden boyutunu döndürür. Bkz. Bir Yığınağın Durumu.

      makro
#include <obstack.h>
int obstack_alignment_mask(struct obstack *obstack-ptr)

Büyüyen nesnenin hizalanmasında kullanılacak maske. Sol taraf değeri olup maske buna atanır. Bkz. Yığınaktaki Verinin Hizalanması.

      makro
#include <obstack.h>
int obstack_chunk_size(struct obstack *obstack-ptr)

Öbeklere tahsis edilecek boyut. Sol taraf değeri olup değer buna atanır. Bkz. Yığınak Öbekleri.

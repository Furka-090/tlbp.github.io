27. Süreçler Arası İletişim

Bu oylumda GNU C Kütüphanesi süreçler arası iletişim ilkelleri açıklanmıştır.

1. Semaforlar

GNU C Kütüphanesi semafor arayüzlerini POSIX ve System V'te tanımlandığı şekilde gerçekler. Semaforlar, paylaşılan kaynakları eşgüdümlemek için birden çok süreç tarafından kullanılabilir. GNU C Kütüphanesi tarafından sağlanan semafor işlevlerinin tam listesi aşağıdadır.

1.1. System V Semaforları

      işlev
#include <sys/types.h>
#include <sys/ipc.h>
#include <sys/sem.h>
int semctl(int semid, int semnum, int cmd, …)

Ön Bilgi

This chapter describes functions for performing low-level input/output operations on file descriptors. These functions include the primitives for the higher-level I/O functions described in Akımlar Üzerinde Giriş/Çıkış, as well as functions for performing low-level control operations for which there are no equivalents on streams.

4.2. Büyük-Küçük Harf Dönüşümleri

Bu kısımda karakterler üzerinde harf büyüklüklerine göre dönüşüm işlemlerinden bahsedilmiştir. Örneğin, toupper işlevi bir karakteri mümkünse büyük harfe dönüştürmek için kullanılır. Eğer karakter dönüştürülemiyosa toupper işlevi karakteri değiştirmez.

Bu işlevler karakteri int türünden bir bağımsız değişken olarak alır ve dönüştürülen karakteri int türünden geri döndürür. Verilen bağımsız değişkene dönüşüm uygulanabilir değilse verilen bağımsız değişken değiştirilmeden döner.

Uyumluluk Bilgisi:

ISO C öncesi oluşumlarda, bu işlevler başarısız olduklarında verilen bağımsız değişken ile dönmez, sadece başarısız olurlardı. Uyumluluk açısından toupper(c) yerine islower(c) ? toupper(c) : c yazmak gerekebilir.

Bu işlevler ctype.h başlık dosyasında bildirilmiştir.

      işlev
#include <ctype.h>
int  tolower(int  c)
Evresel Güvenilirlik: | MT-Safe | AS-Safe | AC-Safe

c büyük harf ise, işlev ona karşılık gelen küçük harf ile döner. c büyük harf değilse işlev hiçbir değişiklik yapmadan c karakteri ile döner.

      işlev
#include <ctype.h>
int  toupper(int  c)
Evresel Güvenilirlik: | MT-Safe | AS-Safe | AC-Safe

c küçük harf ise, işlev ona karşılık gelen büyük harf ile döner. c küçük harf değilse işlev hiçbir değişiklik yapmadan c karakteri ile döner.

      işlev
#include <ctype.h>
int  toascii(int  c)
Evresel Güvenilirlik: | MT-Safe | AS-Safe | AC-Safe

Bu işlev karakterin yüksek seviyeli bitini temizleyerek ASCII karakter kümesindeki 7 bitlik unsigned char türünden karşılığını döndürür. Bu işlev BSD ve SVID oluşumudur.

      işlev
#include <ctype.h>
int  _tolower(int  c)
Evresel Güvenilirlik: | MT-Safe | AS-Safe | AC-Safe

Bu işlev tolower işlevinin eşdeğeri olup SVID ile uyumluluk için bulunmaktadır.

#include <ctype.h>
int  _toupper(int  c)
Evresel Güvenilirlik: | MT-Safe | AS-Safe | AC-Safe

Bu işlev toupper işlevinin eşdeğeri olup SVID ile uyumluluk için bulunmaktadır.

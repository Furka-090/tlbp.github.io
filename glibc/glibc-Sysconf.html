32.4. sysconf Kullanımı

Sistem, yapılandırılabilen sistem sınırlarına sahipse, herhangi bir makinede bu değerler sysconf işlevi ile saptanabilir. Bu işlev ve bu işlevle ilgili parameter sabitleri unistd.h başlık dosyasında bildirilmiştir.

32.4.1. sysconf Tanımı

      işlev
#include <unistd.h>
long int sysconf(int parameter)
Evresel Güvenilirlik: | MT-Safe env | AS-Unsafe lock heap | AC-Unsafe lock mem fd

Bu islev çalışma anında sistem değişkenlerini sorgulamak için kullanılır. parameter bağımsız değişkeni olarak aşağıda listelenen _SC_ simgelerinden biri kullanılır.

İşlevin normal dönüş değeri işlevden istenen değerdir. -1 değeri dönmüşse bu gerçeklenimin böyle bir sınır içermediğini belirtebileceği gibi bir hata durumunu da gösterebilir.

Aşağıdaki errno hata durumu bu işlev için tanımlanmıştır:

EINVAL

parameter değeri geçersiz.

_SC_SIGSTKSZ

sysconf işlevi, bir sinyal yığıtı için gerekli yığıt alanının önerilen asgari bayt sayısı ile ilgili bilgi döndürür.

Bunun, tek, iç içe olmayan, boş bir işleyicinin çağrılması dışında herhangi bir özel amaç için yeterli olduğu garanti edilmez, ancak yine de basit sinyal işleyicileri ve çok düşük düzeyde (örneğin, 2 veya en fazla 3 seviye) iç içe sinyal içeren temel senaryolar için yeterli olmalıdır.

Bu değer, geliştiriciye kolaylık sağlamak ve eski SIGSTKSZ sabitinden geçişi kolaylaştırmak için sağlanmıştır. Daha güçlü garantiler gerektiren yazılımlarda, mümkünse bunu kullanmaktan kaçınılmalıdır.

32.4.3. sysconf Örnekleri

Önce ilgili bağımsız değişken için makro tanımı sınandıktan sonra, yalnızca makro tanımlı değilse sysconf çağrısı yapılmalıdır. Örneğin, burada sistemde iş denetimi desteği olup olmadığı sınanmaktadır: 

int
have_job_control (void)
{
#ifdef _POSIX_JOB_CONTROL
  return 1;
#else
  int value = sysconf (_SC_JOB_CONTROL);
  if (value < 0)
    /* Sistem hata verdiğine göre denenecek
       başka bir şey kalmadı.  */
    fatal (strerror (errno));
  return value;
#endif
}

Burada ise bir sayısal sınırın değeri alınmaktadır: 

int
get_child_max ()
{
#ifdef CHILD_MAX
  return CHILD_MAX;
#else
  int value = sysconf (_SC_CHILD_MAX);
  if (value < 0)
    fatal (strerror (errno));
  return value;
#endif
}

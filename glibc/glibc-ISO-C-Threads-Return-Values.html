35.1.1. Dönen Değerler

ISO C evre belirtimi, arayüzdeki işlevlerin dönüş değerleri için aşağıdaki sayılama sabitlerini sağlar:

      sabit
#include <threads.h>
thrd_timedout

Genellikle karşılıklı dışlayıcı veya durum değişkeni olan istenen kaynak alınmadan belirtilen zamana ulaşıldı.

      sabit
#include <threads.h>
thrd_success

İstenen işlem başarılı oldu.

      sabit
#include <threads.h>
thrd_busy

İstenen öz kaynak kullanımda olduğundan istenen işlem başarısız oldu.

      sabit
#include <threads.h>
thrd_error

İstenen işlem başarısız oldu.

      sabit
#include <threads.h>
thrd_nomem

Yeterli bellek tahsis edilemediğinden istenen işlem başarısız oldu.

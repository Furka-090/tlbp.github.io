13.15.4. Dosya Durum Seçeneklerinin Saptanması

Dosya durum seçeneklerini öğrenmek ve belirlemek için fcntl işlevi kullanılır.

      makro
#include <fcntl.h>
int F_GETFL

Bu makro fcntl işlevinin command bağımsız değişkeninde kullanıldığında filedes tanıtıcı kimliği ile ilgili açık dosyanın dosya durum seçeneklerini okumak için kullanılır.

Bu komutun fcntl işlevinden döndürdüğü değer normalde seçeneklerin bitsel VEYA'lanmasıyla elde edilen negatif olmayan bir değerdir. Dosya erişim kipleri tek bitlik değerler olmadıklarından bu bitlere O_ACCMODE makrosu ile elde edilen maske kullanılarak erişilebilir.

Bir hata durumunda fcntl -1 ile döner. Aşağıdaki errno hata durumu bu komut için tanımlanmıştır:

EBADF

filedes bağımsız değişkeni geçersiz.

      makro
#include <fcntl.h>
int F_SETFL

Bu makro fcntl işlevinin command bağımsız değişkeninde kullanıldığında filedes tanıtıcı kimliği ile ilgili açık dosyanın dosya durum seçeneklerini belirtmek için kullanılır. Bu komut, yeni seçeneklerin belirtildiği int türünde üçüncü bir bağımsız değişkenin varlığını gerektirir: 

fcntl (tanıtıcı-kimliği, F_SETFL, yeni_seçenekler)

Bir dosya tanıtıcının erişim kipi bu komutla değiştirilemez. Dosya tanıtıcısı yazmak ya da okumak için açılmıştır ve artık bu değişmez.

Bu komutun fcntl işlevinden döndürdüğü değer hata durumunu belirten -1 değeri dışında belirsizdir. Hata durumları F_GETFL komutu ile aynıdır.

Örnek:

Dosya durum seçenekleri değiştirilmek istenirse önce geçerli seçenekler F_GETFL ile alınmalı sonra bu değer değiştirilmelidir. Sadece burada açıklanan seçeneklerin var olduğu gibi bir kabul yapılmamalıdır; yazılımı yıllar sonra daha fazla seçeneğin var olduğu bir sistem üzerinde de çalışabilmelidir. Örneğin, aşağıdaki işlev diğer seçeneklere dokunmadan sadece O_NONBLOCK seçeneğini değiştirir: 

/* desc kimliğine O_NONBLOCK seçeneği,
  value sıfırdan farklıysa atanır, değilse temizlenir.
  Dönüş değeri hata yoksa 0, varsa -1 olur
  ve hata errno'ya atanır. */

int
set_nonblock_flag (int desc, int value)
{
  int oldflags = fcntl (desc, F_GETFL, 0);
  /* seçeneklerin okunması başarısız olursa,
     hata durmunun belirtip hemen dön. */
  if (oldflags == -1)
    return -1;
  /* Seçeneğin durumunu belirlenebilir */
  if (value != 0)
    oldflags |= O_NONBLOCK;
  else
    oldflags &= ~O_NONBLOCK;
  /* Değiştirilen seçeneği tanıtıcı kimliğine ata. */
  return fcntl (desc, F_SETFL, oldflags);
}
